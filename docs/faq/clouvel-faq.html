1 설치 문제

pip install clouvel 이 안 돼요

원인: Python 버전이 낮거나 pip가 오래됨

해결:

# Python 버전 확인 (3.10 이상 필요)

python --version

# pip 업그레이드

pip install --upgrade pip

# 다시 설치

pip install clouvel

Claude Code에서 MCP 서버가 안 보여요

원인: claude.json 설정이 안 됨

해결:

# ~/.claude.json 파일 확인

{

"mcpServers": {

"clouvel": {

"command": "python",

"args": ["-m", "clouvel"]

}

}

}

설정 후 Claude Code 재시작 필수!

Windows에서 경로 문제가 생겨요

원인: Python 경로에 공백이나 한글이 포함됨

해결:

# Python 전체 경로 사용

"command": "C:\\Users\\사용자명\\AppData\\Local\\Programs\\Python\\Python311\\python.exe"

⚠️ 경로에 한글이 있으면 영문 경로로 Python 재설치 권장

2 라이선스 활성화

라이선스 키는 어디서 받나요?

구매 후 이메일로 전송됩니다.

형식: CLOUVEL-XXXX-XXXX-XXXX

못 받았으면 스팸함 확인 또는 크몽 메시지로 문의하세요.

라이선스 활성화 방법

Claude Code에서 다음 명령어 실행:

> activate_license 도구 사용

> license_key: "CLOUVEL-XXXX-XXXX-XXXX"

✓ "라이선스 활성화됨" 메시지가 나오면 성공!

"이미 활성화된 라이선스" 에러

원인: 다른 PC에서 이미 활성화됨

해결:

  • Personal: 1대만 가능 → 기존 PC 비활성화 필요
  • Team 5: 5대까지 가능
  • Team 10: 10대까지 가능

PC 교체가 필요하면 크몽 메시지로 문의하세요.

라이선스 상태 확인 방법

> license_status 도구 사용

# 결과 예시

티어: Personal | 상태: 활성 | 만료: 없음

3 can_code 차단될 때

"PRD가 없습니다" 메시지

원인: docs/PRD.md 파일이 없음

해결:

# start 도구로 PRD 생성

> start 도구 사용

> path: "/프로젝트/경로"

또는 직접 docs/PRD.md 파일 생성

"PRD에 필수 섹션이 없습니다"

원인: PRD.md에 필수 항목 누락

필수 섹션:

  • ## 프로젝트 개요
  • ## 핵심 기능
  • ## 기술 스택 (또는 ## 기술 요구사항)

위 섹션이 PRD.md에 있는지 확인하세요.

can_code를 일시적으로 우회하고 싶어요

방법: DEV_MODE 환경변수 설정

# ~/.claude.json

{

"mcpServers": {

"clouvel": {

"command": "python",

"args": ["-m", "clouvel"],

"env": {

"CLOUVEL_DEV_MODE": "true"

}

}

}

}

⚠️ 개발/테스트 용도로만 사용하세요. 프로덕션에서는 끄세요.

4 PRD 작성 팁

PRD 기본 템플릿
# 프로젝트명 ## 프로젝트 개요 - 한 줄 설명: [프로젝트가 뭔지] - 목표: [달성하려는 것] - 타겟 사용자: [누가 쓸 건지] ## 핵심 기능 1. 기능 A - [설명] 2. 기능 B - [설명] 3. 기능 C - [설명] ## 기술 스택 - Frontend: React, TypeScript - Backend: Node.js, Express - Database: PostgreSQL ## 제약 조건 - [제약 1] - [제약 2]
PRD 자동 생성하기

Claude Code에서 start 도구 사용:

> start 도구 사용

> path: "/프로젝트/경로"

# Claude가 질문을 던지면 답변하세요

# 대화가 끝나면 PRD.md가 자동 생성됩니다

좋은 PRD vs 나쁜 PRD

❌ 나쁜 예:

## 핵심 기능

- 로그인

- 게시판

✓ 좋은 예:

## 핵심 기능

1. 로그인

- 이메일/비밀번호 인증

- 소셜 로그인 (Google, GitHub)

- JWT 토큰 기반 세션

2. 게시판

- CRUD (생성, 조회, 수정, 삭제)

- 마크다운 에디터

- 이미지 업로드 (최대 5MB)

구체적일수록 Claude가 더 정확하게 코딩합니다.

6 에러 해결

"MCP server not responding"

원인: Clouvel 서버가 시작 안 됨

해결:

  1. Claude Code 완전 종료 후 재시작
  2. 터미널에서 python -m clouvel 직접 실행해서 에러 확인
  3. 에러 메시지 확인 후 해결
"Network error" / "Connection refused"

원인: 라이선스 서버 연결 실패

해결:

  • 인터넷 연결 확인
  • 방화벽/VPN 확인
  • 잠시 후 재시도

계속 안 되면 크몽 메시지로 문의하세요.

"Invalid license key"

원인: 라이선스 키 오타 또는 만료

해결:

  • 키 복사/붙여넣기 (직접 입력 X)
  • 앞뒤 공백 제거
  • 형식 확인: CLOUVEL-XXXX-XXXX-XXXX
