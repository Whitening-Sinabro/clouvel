# Clouvel Landing Page - Layout Options

> 레퍼런스 기반 10가지 레이아웃 옵션

---

## 레퍼런스 출처

- Linear, Raycast, Supabase, Vercel, Cursor
- Cloudflare Workers, Warp, Arc, xmcp, Poolside

## Option 1: Linear Style (다크 + 퍼플 그라데이션)

**영감**: Linear

**특징**:

- 다크 배경 (#0F0F0F)
- 퍼플 그라데이션 액센트
- Inter 폰트
- 중앙 정렬 Hero
- 미묘한 글로우 효과

**레이아웃**:

```
┌─────────────────────────────────────┐
│  Nav: 로고 ─────────── Links  CTA  │
├─────────────────────────────────────┤
│                                     │
│         [퍼플 그라데이션 뱃지]        │
│                                     │
│      PRD 없으면, 코딩 없다.          │
│         (퍼플 그라데이션 텍스트)       │
│                                     │
│      서브 카피 한 줄               │
│                                     │
│      [ Get Started ]  [ Demo ]     │
│                                     │
├─────────────────────────────────────┤
│  Features: 2x2 그리드, 글로우 카드   │
├─────────────────────────────────────┤
│  Code Demo: 터미널 스타일 블록       │
├─────────────────────────────────────┤
│  Pricing: 다크 카드                 │
└─────────────────────────────────────┘
```

**색상**: `#0F0F0F` `#7C3AED` `#A855F7` `#FFFFFF`
