FAQ

특정 PRD 형식을 강제하나요?

템플릿을 제공하지만 강제하지 않습니다. 필수 섹션(목표, 범위, 요구사항)만 있으면 됩니다.

제 PRD 템플릿을 사용할 수 있나요?

네. 기존 템플릿을 그대로 사용하세요. Clouvel은 특정 형식이 아닌 문서 "존재"만 확인합니다.

오프라인/로컬에서 작동하나요?

라이선스 활성화는 인터넷이 한 번 필요합니다. 그 후에는 완전히 오프라인으로 작동합니다.

기존 워크플로우에 어떻게 맞나요?

MCP 서버입니다 - Claude Code/Desktop에 설정만 추가하면 됩니다. 코드나 프로젝트 구조를 건드리지 않습니다.

그냥 프롬프트나 CLAUDE.md를 쓰면 안 되나요?

프롬프트는 한 세션에서 살고, 메모리는 영원히 삽니다. CLAUDE.md는 정적 규칙이고, Clouvel은 실제 결정에서 배우고 같은 실수를 반복하려 할 때 경고합니다.
