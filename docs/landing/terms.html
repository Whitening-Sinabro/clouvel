1. 서비스 개요

Clouvel("서비스")은 SINABRO("회사")가 제공하는 MCP(Model Context Protocol) 서버 및 관련 도구입니다. 본 서비스는 Claude Code와 통합되어 문서 기반 개발 프로세스를 지원합니다.

2. 라이선스

Clouvel은 다음 두 가지 플랜으로 제공됩니다:

  • Free: 10개 도구, 무료, 영구 사용 가능
  • Pro: 20개 도구 (Free 10개 + Pro 10개), 1명의 사용자, 1개 디바이스, $7.99/월 또는 $49/년

Pro 라이선스는 양도, 재판매, 공유가 금지됩니다. 위반 시 라이선스가 취소될 수 있습니다.

3. 면책 조항

중요: 본 서비스는 "있는 그대로(AS IS)" 제공됩니다.

  • 회사는 서비스의 특정 결과나 성능을 보장하지 않습니다.
  • 랜딩페이지에 표시된 수치(디버깅 시간 감소, 스코프 크립 차단 등)는 특정 프로젝트 환경에서의 결과이며, 모든 사용자에게 동일한 결과를 보장하지 않습니다.
  • 서비스 사용으로 인한 직접적, 간접적, 우발적, 특수적, 결과적 손해에 대해 회사는 책임을 지지 않습니다.
  • 서비스는 코드 품질 향상을 위한 도구일 뿐, 버그 없는 소프트웨어를 보장하지 않습니다.

4. 환불 정책

디지털 제품의 특성상, 구매 후 14일 이내에 환불을 요청할 수 있습니다. 단, 라이선스 키가 활성화된 경우 환불이 제한될 수 있습니다. 환불 요청은 GitHub Issues를 통해 접수해주세요.

5. 업데이트 및 지원

Pro 라이선스에는 구독 기간 동안의 모든 업데이트가 포함됩니다. 기술 지원은 GitHub Issues 및 문의 양식을 통해 제공되며, 일반적으로 24~48시간 내에 응답합니다 (best effort 기준, 법적 SLA는 아닙니다).

6. 서비스 변경

회사는 사전 통지 없이 서비스의 기능을 변경, 추가 또는 제거할 수 있습니다. 중대한 변경 사항은 GitHub 및 이메일을 통해 공지됩니다.

7. 약관 변경

본 약관은 필요에 따라 변경될 수 있습니다. 변경된 약관은 본 페이지에 게시되며, 게시 후 서비스를 계속 사용하면 변경된 약관에 동의한 것으로 간주됩니다.

8. 준거법

본 약관은 대한민국 법률에 따라 해석되며, 관련 분쟁은 대한민국 법원의 관할에 따릅니다.
