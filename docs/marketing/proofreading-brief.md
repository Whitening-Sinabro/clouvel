# English Proofreading Brief

## For Fiverr/Upwork

### Project Overview
Native-level English proofreading for a developer tool landing page.

### Target Audience
- Solo developers (indie hackers)
- Team leads managing junior developers
- English-speaking, technical background
- Familiar with AI coding tools

### Tone & Style
- Professional but approachable
- Direct, no fluff
- Technical accuracy matters
- Conversational where appropriate

### File to Review
`i18n/en.json` - JSON file with all English copy

### What We Need
1. **Grammar & spelling fixes**
2. **Natural phrasing** - Does it sound native?
3. **Consistency** - Same terms throughout
4. **Clarity** - Is the message clear?

### Key Terms (Don't Change)
- PRD (Product Requirements Document)
- MCP (Model Context Protocol)
- Claude Code (product name)
- Clouvel (our product name)

### Sections Priority

| Section | Priority | Notes |
|---------|----------|-------|
| hero | HIGH | First impression |
| problem | HIGH | Pain point messaging |
| features | HIGH | Value proposition |
| pricing | MEDIUM | Clear pricing copy |
| faq | MEDIUM | Common questions |
| contact | LOW | Standard form labels |

### Deliverable
- Corrected JSON file OR
- Google Doc with tracked changes

### Timeline
2-3 business days

### Budget
$30-50 USD
