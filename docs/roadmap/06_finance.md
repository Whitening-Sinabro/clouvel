```
Free ($0):
├── 5개 활성 스펙

Pro ($15/월):
├── 무제한 스펙

```

### LTV 계산

```
Pro 고객 LTV:
- 월 수익: $15
- 평균 유지 기간: 18개월
- LTV = $15 × 18 = $270

Team 고객 LTV:
- 월 수익: $12 × 5명 = $60
- 평균 유지 기간: 24개월
- LTV = $60 × 24 = $1,440

Enterprise 고객 LTV:
- 월 수익: $50 × 60명 = $3,000
- 평균 유지 기간: 36개월
- LTV = $3,000 × 36 = $108,000
```

## KPIs

### North Star Metric

```
"스펙 기반으로 성공적으로 완료된 프로젝트 수"

왜?
- 핵심 가치 직접 반영
- 사용자 성공과 연결
- 성장과 상관관계
```

### 단계별 핵심 KPI

| 단계 | 핵심 KPI             | 목표     | 왜?            |
| ---- | -------------------- | -------- | -------------- |
| 2026 | 가입자, MAU          | 10K / 3K | 초기 채택 증명 |
| 2027 | GitHub Stars, 템플릿 | 5K / 500 | 커뮤니티 성장  |
| 2028 | ARR, 기업 고객       | $6M / 50 | 수익성 증명    |

## 펀딩 시나리오

### 시나리오 A: 부트스트랩

```
가정:
- 외부 투자 없음
- 수익으로만 성장
- 느리지만 안정적

타임라인:
2026: $120K ARR, 손익분기
2027: $1.2M ARR, 소규모 팀 (3-5명)
2028: $6M ARR, 중규모 팀 (10-15명)
2029+: 지속 성장 또는 전략적 M&A

장점:
- 100% 지분 유지
- 압박 없는 의사결정
- 지속 가능한 성장

단점:
- 느린 성장
- 경쟁자에게 시장 빼앗길 위험
- 대형 기능 개발 제한
```

### 시나리오 B: 시드 라운드

```
가정:
- 2027년 시드 $1-3M
- ARR $1M+ 시점
- 팀 확장에 집중

타임라인:
2027: 시드 $2M @ $10M 밸류
2028: $6M ARR
2029: 시리즈 A $10M @ $50M 밸류
2030: $48M ARR 또는 Exit

장점:
- 빠른 팀 확장
- 마케팅 투자 가능
- 경쟁 대응력

단점:
- 지분 희석 (20-30%)
- VC 압박
- Exit 기대
```

### 시나리오 C: 전략적 투자

```
가정:
- Anthropic, Cursor 등 전략적 투자자
- 기술/비즈니스 시너지
- Exit 경로 명확

타임라인:
2027-2028: 전략적 투자 $5M
2029-2030: 인수 또는 추가 투자

장점:
- 기술 협력
- 배포 채널 확보
- Exit 보장에 가까움

단점:
- 독립성 제한
- 특정 플랫폼 종속
- 경쟁사 배제
```
